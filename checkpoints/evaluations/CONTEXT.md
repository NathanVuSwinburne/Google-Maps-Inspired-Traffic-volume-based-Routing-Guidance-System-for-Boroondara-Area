# checkpoints/evaluations — Context

## Purpose
Contains **test-set evaluation outputs** for each model run: scalar metrics, scatter plots, and per-location prediction time-series examples.

## File Layout (per run)

```
evaluations/{run_id}/
├── figures/
│   ├── {run_id}_scatter.png              # Actual vs Predicted scatter plot
│   ├── {run_id}_error_hist.png           # Prediction error distribution histogram
│   └── {run_id}_{location}_*_example.png # Time-series for sampled locations
```

## Metrics Reported
| Metric | Description |
|--------|-------------|
| `MSE` | Mean Squared Error |
| `RMSE` | Root Mean Squared Error — primary ranking metric |
| `MAE` | Mean Absolute Error |
| `R²` | Coefficient of Determination |

## Generated By
`src/train_and_evaluate/train_and_evaluate.py::evaluate_model()` — called automatically after each training run.

## Interpretation Guide
- **Scatter plot**: points on the red dashed line = perfect prediction. Spread indicates variance of error.
- **Error histogram**: centred around 0 is ideal; skew indicates systematic over/under-prediction.
- **Time-series examples**: up to 288 time-steps (3 days × 96 intervals) for 10 randomly sampled locations. Shows how well the model tracks peak and off-peak patterns.

## Usage
These artefacts are for **offline model comparison** only. They are not loaded or served by the Streamlit app at runtime.
